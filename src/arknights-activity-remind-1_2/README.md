#### 本插件依赖【明日方舟数据解析】插件，且需要及时更新游戏数据才能保证准时提醒
#### 支持提醒活动开始、活动结束、活动奖励兑换结束。
- 仅支持游戏内活动的提醒

- 由管理员开启活动提醒
    - 发送`兔兔开启活动提醒`以开启本群或本频道活动提醒推送
    - 发送`兔兔关闭活动提醒`以关闭本群或本频道活动提醒推送

- 发送`兔兔活动列表`可查看近期活动及卡池的开启关闭信息

#### 配置

- 前往控制台 >> 插件管理 >> 插件配置，管理提醒推送设置
